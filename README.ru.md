# 16 Ассоциаций: Самоанализ

Бессерверное одностраничное JavaScript-приложение (SPA) для прохождения теста «16 ассоциаций» — психологической методики самоанализа. Приложение полностью работает в браузере без серверной обработки и передачи данных.

## Обзор

Тест «16 ассоциаций» — это психологический самоанализ, который помогает пользователям исследовать свои бессознательные ассоциации и мыслительные паттерны, анализируя их мгновенные реакции на 16 различных слов. Эта реализация позволяет проходить тест офлайн, при этом все данные остаются в вашем браузере.

## Особенности

- ✅ **Бессерверный**: Полностью работает в браузере без серверной части
- ✅ **Фокус на конфиденциальности**: Никакие данные не отправляются на сервер
- ✅ **Офлайн-совместимость**: Работает без подключения к интернету
- ✅ **Адаптивный дизайн**: Работает на мобильных устройствах, планшетах и компьютерах
- ✅ **Инструмент самоанализа**: Помогает исследовать бессознательные ассоциации и инсайты

## Технические детали

Это одностраничное приложение, созданное с использованием современных веб-технологий:
- HTML5 для структуры
- CSS3 для стилизации
- JavaScript для интерактивности
- Нет внешних зависимостей или фреймворков

## Как использовать

1. Откройте приложение в любом современном веб-браузере
2. Ознакомьтесь с инструкцией теста «16 ассоциаций»
3. Введите свои ассоциации для каждого из 16 слов
4. Просмотрите результаты и инсайты
5. Все данные остаются в вашем браузере — никакая информация не передается

## Заявление о конфиденциальности

Это приложение полностью бессерверное. Никакие данные не отправляются на сервер, не сохраняются и не обрабатываются за пределами вашего браузера. Приложение разработано с учетом максимальной конфиденциальности.

## О тесте «16 ассоциаций»

Тест «16 ассоциаций» — это психологическая методика, использующая словесные ассоциации для получения инсайтов о бессознательных мыслях и эмоциональных паттернах человека. Участникам предлагается мгновенно реагировать на список из 16 слов, предоставляя первое приходящее в голову слово, что позволяет исследовать паттерны ассоциативного мышления.
